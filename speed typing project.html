MY SPEED TYPING PROJECT


Over the past week I have been using my knowledge of programming in the boot camp to work on my own project.  My project idea was a speed typing game where the user had to input a word that was randomly generated for them from an array.  After inputting the word they would hit enter.  If the word was correct they would receive a point.  If they misspelled the word a point would be taken away.


This project has allowed me to make use of

  • global and local variables
  • external files
  • random number generators
  • arrays
  • user input
  • colours
  • time

Bugs

There were also bugs in my code which needed fixing, the biggest issue I wasn't able to resolve was that the timer would count down from 60 seconds as intended but it wouldn't update each second.  Instead it would only update every time a user pressed a key.  The timer also started going into the negatives after it reached 0.  

I coded a message to be displayed that would tell the user that time is up, the message displayed correctly however the program doesn't stop running after time is up.
